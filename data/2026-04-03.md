### [2] [PI-JEPA: Label-Free Surrogate Pretraining for Coupled Multiphysics Simulation via Operator-Split Latent Prediction](https://arxiv.org/abs/2604.01349)
*Brandon Yee, Pairie Koh*

Main category: cs.LG

TL;DR: PI-JEPA是一种无需完整PDE求解的标签自由预训练框架，通过对无标签参数场的掩码潜变量预测和残差正则化，显著降低了多物理场模拟代理模型的训练成本。


<details>
  <summary>Details</summary>
Motivation: 传统神经网络代理模型需要大量的标注模拟轨迹进行训练，而实际应用中获取这些数据成本高昂。存在数据不对称性：可以生成大量无标签的参数场，但现有模型无法利用这些数据。

Method: PI-JEPA采用物理信息联合嵌入预测架构，利用Lie-Trotter算子分裂将PDE分解为多个子过程，为每个子过程分配独立的物理约束潜变量模块。首先在大量无标签参数场上进行预训练，然后仅使用少量标注模拟数据进行微调。

Result: 在单相达西流模拟中，PI-JEPA在Nℓ=100时，误差比FNO低1.9倍，比DeepONet低2.4倍，在Nℓ=500时，比仅进行监督训练的性能提升了24%。

Abstract: Reservoir simulation workflows face a fundamental data asymmetry: input parameter fields (geostatistical permeability realizations, porosity distributions) are free to generate in arbitrary quantities, yet existing neural operator surrogates require large corpora of expensive labeled simulation trajectories and cannot exploit this unlabeled structure. We introduce \textbf{PI-JEPA} (Physics-Informed Joint Embedding Predictive Architecture), a surrogate pretraining framework that trains \emph{without any completed PDE solves}, using masked latent prediction on unlabeled parameter fields under per-sub-operator PDE residual regularization. The predictor bank is structurally aligned with the Lie--Trotter operator-splitting decomposition of the governing equations, dedicating a separate physics-constrained latent module to each sub-process (pressure, saturation transport, reaction), enabling fine-tuning with as few as 100 labeled simulation runs. On single-phase Darcy flow, PI-JEPA achieves $1.9\times$ lower error than FNO and $2.4\times$ lower error than DeepONet at $N_\ell{=}100$, with 24\% improvement over supervised-only training at $N_\ell{=}500$, demonstrating that label-free surrogate pretraining substantially reduces the simulation budget required for multiphysics surrogate deployment.

### [3] [Gradient estimators for parameter inference in discrete stochastic kinetic models](https://arxiv.org/abs/2604.02121)
*Ludwig Burger, Annalena Kofler, Lukas Heinrich, Ulrich Gerland*

Main category: physics.comp-ph

TL;DR: 本文研究了如何将梯度估计应用于离散随机动力学模型中的参数推断，并比较了三种机器学习中的梯度估计器在吉尔伯特随机模拟算法(SSA)中的性能。


<details>
  <summary>Details</summary>
Motivation: 在物理学中，随机动力学模型无处不在，但从实验数据中推断其参数具有挑战性。传统的基于梯度的参数推断方法无法直接应用于吉尔伯特SSA，因为SSA涉及不可微的离散采样操作。

Method: 本文采用了三种机器学习中的梯度估计器（Gumbel-Softmax Straight-Through (GS-ST) 估计器、Score Function 估计器和 Alternative Path 估计器）应用于吉尔伯特SSA，并在两个具有弛豫或振荡动态的代表性系统中比较了这些估计器的性能。

Result: 研究发现，GS-ST估计器通常能产生行为良好的梯度估计，但在某些参数条件下，其方差会发散，导致参数推断失败。在这些情况下，其他估计器提供了更稳健、方差更低的梯度。

Conclusion: 本文表明，基于梯度的参数推断可以有效地与吉尔伯特SSA集成，不同的估计器具有互补的优势，为随机动力学模型的参数推断提供了一种新的方法。

Abstract: Stochastic kinetic models are ubiquitous in physics, yet inferring their parameters from experimental data remains challenging. In deterministic models, parameter inference often relies on gradients, as they can be obtained efficiently through automatic differentiation. However, these tools cannot be directly applied to stochastic simulation algorithms (SSA) such as the Gillespie algorithm, since sampling from a discrete set of reactions introduces non-differentiable operations. In this work, we adopt three gradient estimators from machine learning for the Gillespie SSA: the Gumbel-Softmax Straight-Through (GS-ST) estimator, the Score Function estimator, and the Alternative Path estimator. We compare the properties of all estimators in two representative systems exhibiting relaxation or oscillatory dynamics, where the latter requires gradient estimation of time-dependent objective functions. We find that the GS-ST estimator mostly yields well-behaved gradient estimates, but exhibits diverging variance in challenging parameter regimes, resulting in unsuccessful parameter inference. In these cases, the other estimators provide more robust, lower variance gradients. Our results demonstrate that gradient-based parameter inference can be integrated effectively with the Gillespie SSA, with different estimators offering complementary advantages.

### [4] [The topological gap at criticality: scaling exponent d + η, universality, and scope](https://arxiv.org/abs/2604.01484)
*Matthew Loftus*

Main category: cond-mat.stat-mech

TL;DR: 本文研究了自旋模型中拓扑间隙与临界态的关系，并建立了一个完整的有限尺寸标度律，揭示了拓扑间隙与标度指数d+η之间的联系，并探讨了该框架的适用范围。


<details>
  <summary>Details</summary>
Motivation: 之前的研究对拓扑间隙的标度指数和G-指数进行了经验性研究，但缺乏统一的理论框架和更广泛的适用性验证。本文旨在建立一个统一的理论，解释已知的标度规律，并将其推广到其他模型，同时明确该框架的适用范围。

Method: 研究人员使用持久同调（PH）方法分析了五种模型（二维Ising模型、二维Potts模型（q=3和q=4）、三维Ising模型、XY模型和二维渗滤模型）的拓扑间隙。通过模拟计算，获得了不同系统尺寸和温度下的拓扑间隙数据，并使用有限尺寸标度律进行拟合，确定了标度指数和G-指数。

Result: 研究结果表明，拓扑间隙的标度指数α与d+η一致，其中η是异常维度。对于二维Ising模型和二维Potts模型（q=3），G-指数γ与1+β/ν一致。研究还明确了该框架的适用范围，发现对于第一类相变、Berezinskii-Kosterlitz-Thouless相变和二维渗滤模型，该框架失效。

Conclusion: 本文建立了一个完整的有限尺寸标度律，揭示了拓扑间隙与临界态之间的关系，并为理解自旋模型中的拓扑现象提供了新的视角。研究结果表明，α=d+η对于具有代数修正的二阶相变有效，而对于具有对数修正的相变则失效，为进一步研究拓扑相变提供了重要的指导。

Abstract: The topological gap $Δ= TP_{H_1}^{real} - TP_{H_1}^{shuf}$ -- the excess $H_1$ total persistence of the majority-spin alpha complex over a density-matched null -- encodes critical correlations in spin models. We establish finite-size scaling: $Δ(L,T) = A L^{d+η} G_-(L|t/T_c|)$, with $G_-(x) \sim (1+x/x_0)^{-(1+β/ν)}$. For 2D Ising, $α= 2.249 \pm 0.038$, matching $d+η= 9/4$ to $0.03σ$; the $G_-$ exponent $γ= 1.089 \pm 0.077$ is consistent with $1+β/ν= 9/8$ ($ΔR^2 < 10^{-5}$). For 2D Potts $q=3$ with $L$ up to 1024, $α= 2.272 \pm 0.024$ ($0.2σ$ from $d+η= 2.267$), with two-term corrections to scaling ($R^2 = 0.9999$). The $G_-$ exponent $γ= 1.114$ (68% CI $[1.053, 1.173]$) matches $1+β/ν= 17/15$. Scope boundaries: the law fails for 2D Potts $q=4$ ($α= 2.347 \pm 0.017$, $9.3σ$ from $d+η= 5/2$) where logarithmic corrections prevent convergence, and for raw 3D Ising ($4σ$ from $d+η$), but density normalization $Δ/|M|^{1/2}$ recovers $α= 3.06 \pm 0.04$ ($0.6σ$). The framework fails for first-order, BKT, and percolation. The criterion: $α= d+η$ holds when corrections to scaling are algebraic ($ω> 0$) but fails when logarithmic ($ω\to 0$).

### [6] [VIANA: character Value-enhanced Intensity Assessment via domain-informed Neural Architecture](https://arxiv.org/abs/2604.01365)

Main category: physics.chem-ph

TL;DR: 本文提出了VIANA框架，通过整合分子结构、语义特征和生物剂量-反应关系，显著提高了嗅觉强度预测的准确性，有效弥合了分子信息学与感官知觉之间的差距。


<details>
  <summary>Details</summary>

This research provides a robust framework for digital olfaction, effectively bridging the gap between molecular informatics and sensory perception.
